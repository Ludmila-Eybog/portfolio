О проекте

Учебный макет для приложения Юнит. Модульный подход при проектировании страницы.
Верстка с использованием препроцессора LESS.
Использованы svg-графика, hover-эффекты.
Адаптивная верстка под планшеты и смартфоны, без использования bootstrap. Перевод на mobile-first.
Верстка сайта согласно макету с использованием Perfect Pixel.
