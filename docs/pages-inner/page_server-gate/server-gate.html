О проекте

Учебный проект для интернет-магазина серверов.
Верстка с использованием препроцессора LESS.
Модульный подход при проектировании страницы.
Адаптация под планшет и смартфон.
Перевод на mobile-first.
Оптимизация изображений для Retina дисплеев.
Использованы svg-графика, hover-эффекты.
Оптимизация графики.
Верстка сайта согласно макету с использованием Perfect Pixel.

Детали
  • Проект:
    интернет-магазин серверов "Server-Gate"
  • Верстка:
    HTML5, CSS3, JS, jQuery, Perfect Pixel
  • Особенности:
    Модель Flex-box, семантический код, BEM-naming методология, кастомизация "всплывающих" меню (шесть окон), 2 выпадающих окна при помощи JS, карусель OWL-Carousel (в трех секциях), 4 точки перелома, кроссбраузерность, медиазапросы на LESS.
